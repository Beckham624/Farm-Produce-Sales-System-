# Farm-Produce-Sales-System-
This Program allows the user to record sales in a farm produce store of which it allows the user register produce,search produce records,update produce details,delete produce records,sell produce, and display sales summaries.
Delete Function
Removes unwanted or outdated produce records from the system.
 Sales Processing
The program:
Sells produce to customers
Reduces stock automatically
Prevents selling beyond available stock
 Bulk Purchase Discount
The system applies discounts automatically when customers buy produce in large quantities.
Example:
Quantity above a certain limit receives a percentage discount.
